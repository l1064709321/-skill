## 七、动态调用流程

1. **分析需求**：识别写作类型（玄幻/都市/悬疑/言情等）、写作要素（开篇、人设、爽点、悬念、对话等）及创作阶段（大纲、正文、修改）。
2. **匹配作家**：从索引中筛选 1-3 位在该领域有突出建树的作家。
3. **提取方法论**：分析该作家风格在文本中的具体体现，提供可实操的创作建议。
4. **给出范例**：提供标志性段落或句式（或模拟文本），并点明精妙之处。

## 八、使用示例

### 示例一：设计玄幻小说主角的“金手指”

**匹配作家**：天蚕土豆、我吃西红柿、爱潜水的乌贼

**创作建议**：
1. **升级体系化 (天蚕土豆)**：设计清晰的修炼等级，金手指提供独特升级路径，但需伴随限制和代价，维持故事张力。
2. **节奏爽快 (我吃西红柿)**：金手指成长与剧情升级节奏紧密挂钩，每次突破都能立即转化为打脸或破局爽点。
3. **融合创新 (爱潜水的乌贼)**：将传统金手指（系统、老爷爷）与新颖概念（克苏鲁、蒸汽朋克）结合，创造反套路独特设定。

### 示例二：写一个让读者恨不起来的反派

**匹配作家**：古龙、曹雪芹、辰东

**创作建议**：
1. **复杂性塑造 (曹雪芹)**：赋予反派独立行为动机和价值观，其“恶”源于自身创伤、执念或极端理念。
2. **独特魅力 (古龙)**：赋予反派一种偏执的“道”或独特的“美学”，如极致的骄傲、扭曲的正义感。
3. **留白与尊重 (辰东)**：给予敌人应有的尊重和体面，通过侧面描写烘托其过往辉煌与悲剧，引发读者共情。

### 示例三：构建克苏鲁元素的东方玄幻世界

**匹配作家**：狐尾的笔、佛前献花、滚开

**创作建议**：
1. **本土化改造 (狐尾的笔)**：将“外神”设定为上一个纪元的“旧日仙尊”，修炼其功法会导致精神污染和异化，但能获得扭曲而强大的力量。
2. **规则化设计 (佛前献花)**：为“污染”或“诡异”设定清晰、可探索的规则（禁忌仪式、不可触碰的“圣物”等），主角需像解谜一样掌握规则来生存。
3. **氛围营造 (滚开)**：在日常描写中埋设违和感与伏笔，如看似普通的村庄却有着共同回避的话题、不自然的习俗或特定日子的集体反常行为。

## 九、半部小说续写引擎

> 写了一二十万字突然卡住了？把半部稿子喂进来，引擎会自动拆解你的作品 DNA，结合大神技法给出能落地的续写方案。

### 9.1 工作流程

1. **吞下稿子**：把你写好的半部小说文本贴进来（建议至少 3 万字）。
2. **AI 解剖**：对稿子进行五个维度拆解——主线与结构、人物档案、风格指纹、伏笔与坑位、爽点密度。
3. **匹配大神外援**：根据作品类型和卡点，从作家索引中自动匹配 1-2 位最对症的大神。
4. **输出续写方案**：给出作品体检报告、2-3 条后续情节路径、可选示范续写段落。

### 9.2 使用方式

直接在对话框输入：
- “我写了一部凡人流修仙文，15 万字卡住了，主角刚结丹成功不知道接下来干嘛，帮我分析一下。”
- “[粘贴小说文本] 帮我看看后面怎么写。”

### 10.2 “去 AI 味”专项清洗

| AI 味表达 | 建议替换方向 |
|:---|:---|
| “他的心中不由得一惊” | 删除“不由得”，直接写动作：“他瞳孔微缩，后退半步。” |
| “这一切说来复杂，实则只发生在电光火石之间” | 直接删掉，用短句推进节奏。 |
| “他知道，自己必须变强” | 换成具体的念头：“再这样下去，下次死的就不是别人了。” |
| “一股暖流涌上心头” | 删掉，用行动代替感受：“他别过脸，没让师父看见自己发红的眼眶。” |
| 每章结尾的“人生感悟” | 删掉，用悬念或动作收尾。 |

### 10.3 文风仿写功能

输入目标文风的参考文本，系统提取句式特征、词汇偏好、叙事视角、节奏指纹，然后按此风格仿写或润色。

### 10.4 审稿报告格式

报告分为：🔴 严重问题（立即修改）、🟡 中等问题（本周期内处理）、🟢 轻微问题（择机优化）、✅ 亮点（保持）。

#### 十一、网文编辑流水线——你的云端码字工作室

> 写书不用自己吭哧吭哧码字。这条流水线把一个网文工作室的整套班底——扫榜的、写大纲的、当枪手的、审稿的、改稿的——全部 AI 化。你只需要当那个拍板的“老板”。

### 11.1 工作室成员

| 角色 | 外号 | 职责 | 输入 | 输出 |
|:---|:---|:---|:---|:---|
| **扫榜小哥** | 市场风向标 | 扒榜、分析趋势 | 起点/番茄榜单、热搜词 | 热门题材报告、脑洞趋势、竞品开篇 |
| **大纲老手** | 框架师 | 世界观与大纲设计 | 扫榜报告 + 你的创意 | 完整设定、分卷大纲、人物小传、伏笔清单 |
| **枪手代笔** | 量产机 | 正文生成 | 大纲 + 指定文风 | 章节正文（2000-4000字/章） |
| **毒舌编辑** | 找茬专家 | 36维质量检查 | 枪手草稿 | 审稿报告（分严重/中等/轻微问题） |
| **改稿编辑** | 救火队员 | 问题修正 | 审稿报告 + 原草稿 | 修订后正文 + 修改说明 |

### 11.2 流水线怎么跑

1. **市场摸底**：**扫榜小哥**出趋势报告 → 你确定要写啥。
2. **搭建骨架**：**大纲老手**根据你的点子出大纲和人设 → 你拍板通过。
3. **批量码字**：**枪手代笔**按大纲批量写稿。
4. **挑刺质检**：**毒舌审稿**对每章进行33维扫描，出审计报告。
5. **返修循环**：如果有严重问题，**改稿编辑**接手修改 → 再送回给**毒舌审稿**，直到问题解决（最多3轮）。
6. **最终拍板**：通过审计的稿子交到你手里，你决定发不发。

### 11.3 你的角色：老板

你只需要在几个关键节点“签字画押”：

| 节点 | 你要干嘛 | 耗时 |
|:---|:---|:---|
| 立项 | 给个点子，批准市场方向 | 10分钟 |
| 大纲确认 | 看看大纲人设，有意见就提，没有就过 | 15-30分钟 |
| 风格定调 | 扔一段你喜欢的文字当参考 | 5分钟 |
| 收货发布 | 审阅成品章节，行就发，不行就打回 | 每章5分钟 |

### 11.4 老板怎么下指令

| 场景 | 指令示例 |
|:---|:---|
| 启动全流程 | “工作室开工！题材：赛博修仙+家族流。主角是背负‘修仙贷’的家族旁系。目标平台起点。” |
| 只要扫榜 | “扫榜小哥，最近一周都市异能有什么新书火了？” |
| 只要大纲 | “大纲老手，给我一份‘诡异修仙’的前三卷大纲，主角设定是能看到死亡倒计时。” |
| 只要审稿 | “毒舌审稿，这一章我总觉得有点水，帮我标一下哪里可以精简。” |
| 文风仿写 | “枪手，分析一下这段《神秘复苏》的描写，然后用这种诡味帮我写个开篇。” |

### 11.5 工作室黑话速查

- **崩了**：战力/逻辑/人设前后矛盾。
- **水了**：章节内容空洞，无剧情推进。
- **毒点**：让读者极度不适的情节（送女、绿帽、圣母等）。
- **AI味**：句式机械、对话客气、结尾总爱升华。
- **钩子**：章节结尾的悬念，勾引读者点下一章。
